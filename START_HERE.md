## ❓ Common Questions

### Q: How do I log in?
A: Click "Login" in the header. Use any email and password (it's a demo). Choose Customer or Admin role.

### Q: Can I change the products?
A: Yes! Edit `src/data/mockData.ts` to customize products and services.

### Q: How do I deploy this?
A: See [DEPLOYMENT.md](DEPLOYMENT.md) for detailed instructions.

### Q: Is this production-ready?
A: The code is production-ready! You'll need to:
1. Replace mock data with real API calls
2. Connect payment gateway
3. Set up authentication
4. Deploy to hosting

### Q: Can I customize the design?
A: Yes! The app uses Tailwind CSS. Edit colors and fonts in `src/index.css`.

### Q: How do I add real payment processing?
A: See [DEPLOYMENT.md](DEPLOYMENT.md) - Payment Gateway Integration section.
