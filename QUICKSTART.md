# Igris - Quick Start Guide

## 🚀 Getting Started

### What is Igris?
Igris is a comprehensive tech gadgets and professional services catalog platform. It's a full-featured e-commerce and service booking application built with React, Vite, and Tailwind CSS.

## 📱 Key Features at a Glance

### For Customers
1. **Browse Products** - Search through 500+ tech gadgets with filters and sorting
2. **Book Services** - Schedule appointments with professional technicians
3. **Shopping Cart** - Add items to cart and proceed to secure checkout
4. **Order Tracking** - Monitor your orders and service bookings
5. **User Profile** - Manage personal information and view purchase history

### For Administrators
1. **Dashboard** - View real-time sales metrics and KPIs
2. **Order Management** - Track and manage customer orders
3. **Catalog Management** - Add, edit, and manage products
4. **Customer CRM** - View customer data and purchase history
5. **Analytics** - Analyze revenue trends and category performance

## 🎯 Main Flows

### 1. Shopping Experience
```
Home → Browse Catalog → Filter/Search Products → View Details → 
Add to Cart → Checkout → Enter Shipping Info → Payment → Order Confirmed
```

### 2. Service Booking
```
Home → Services → Browse Services → Book Now → Select Date/Time → 
Confirm Booking → Appointment Confirmed
```

### 3. Order Management
```
Login → Orders → View Orders/Appointments → Track Status → 
Cancel/Reschedule (if applicable)
```

## 💳 Checkout Process

### For Products
1. Add items to cart
2. Click "Proceed to Checkout"
3. Enter shipping address (name, email, address, city, ZIP, country)
4. Enter payment details (card number, expiry, CVV)
5. Confirm order
6. See order confirmation with tracking info

### For Services
1. Select service
2. Click "Book Now"
3. Choose preferred date and time
4. Confirm booking
5. Receive appointment confirmation
